CBL Project

A web-based DOS emulator running the CBL program. This project uses Em-DOSBox to bring classic DOS applications to modern web browsers.

How to Use

  • Click "Start Program" to begin running the CBL program
  • Use the "Reset" button to restart the emulator if needed
  • The program will run in the black box above
  • Keyboard input is supported for interacting with the program
